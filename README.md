# tjstodbot-help
코딩 교육 플랫폼 엔트리에서 작동하는 tjstod봇의 도움말입니다.

## 소개
[tjstod봇](http://naver.me/5X9C7i11 "마이페이지")은 [tjstod](http://naver.me/ForzBSln "마이페이지")가 만든 봇으로, 여러가지 언어와 디버깅을 배우기 위해 만든 봇입니다.

## 기능
tjstod봇에는 현재도 작동중이거나 지원종료된 다양한 기능이 있습니다.
